Features

The Flow LLM Proxy offers a robust set of features to enhance your LLM integration:

  • Secure Authentication: Utilizes JWT-based authentication with Personal Access Token (PAT) generation.

  • Multi-Provider Support: Seamlessly integrates with various LLM providers, including:

    • OpenAI

    • AWS Bedrock (Claude)

    • Google Gemini

    • DeepSeek

  • Efficient Token Handling: Features thread-safe token caching for improved performance.

  • Reliability: Includes proper error handling and logging mechanisms.

How to Configure

This section guides you through setting up and configuring the Flow LLM Proxy.

There are 3 ways to authenticate in Flow LLM Proxy

1 - Using your session FlowToken

2 - Using your pré-authenticated PAT Token

3 - Generating a JWT Token (recommended)

Below, you can see how to generate your JWT Token

Generate JWT Token

A JSON Web Token (JWT) is required for authenticating with the Flow LLM Proxy.

Step 1: Generate Flow API Key (Secrets)

Before generating a JWT, you need to obtain your clientId, clientSecret, and tenant from the Flow portal.

  1. Go to flow.ciandt.com.

  2. Click on the Avatar Icon (usually in the top right corner).

  3. Select Settings.

  4. Go to API Keys.

  5. Click Create new Key.

  6. Provide a name for your key.

  7. For APPS, select llm-api.

  8. Save these credentials securely.

Step 2: Choose a Method for Token Generation

You can generate your JWT manually or using a script.

For Manual Token Generation:

  1. Visit jwt.io in your web browser.

  2. Select the JWT Encoder option.

  3. Configure the token using the following settings:

    • Headers (Algorithm and Type):

      { 
   "alg": "HS256", 
   "typ": "JWT" 
}

    • Payload (Your Credentials): Replace <your-client-id>, <your-client-secret>, and <your-tenant> with the secrets you obtained from the Flow portal.

      { 
   "clientId": "<your-client-id>", 
   "clientSecret": "<your-client-secret>", 
   "tenant": "<your-tenant>" 
}

  4. The generated JWT will appear on the right side of the page. Copy this token for use in your client configurations.

For Script Generation:

  1. Download the provided script: generate_jwt.sh.

  2. Make the script executable and run it:

    chmod a+x scripts/generate_jwt.sh ./scripts/generate_jwt.sh

  3. The script will output the generated JWT. Copy this token.
